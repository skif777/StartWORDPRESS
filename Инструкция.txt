1--ПОДКЛЮЧЕНИЕ ЛОКАЛЬНОГО ДОМЕНА
   - Создать папку домена
   1.1 OpenServer
       1.1 Перейти в настройки/домены
       1.2 Внести имя домена и выбрать папку, нажать добавить
       1.3 Имя домена добавить в gulpfile.js в таске browser-sync (proxy: "имя домена")

2--УСТАНОВКА
   1. Устанавливаем WP (Распаковывем WP в папку домена).
   2. Устанавливаем тему (В папку wp-content/themes распаковываем стартовый шаблон underscores).
   3. В установленную тему размещаем стартовый шаблон StartWORDPRESS.
   4. Cоздать базу данных (UTF8_general_ci).
   5. Установить WP (Ввести имя базы, пароль базы, имя пользователя - root).
   6. Активировать в ПУ установленную тему (внешний вид/темы).
   8. Установить Unison, активировать и загрузить конструктор страниц(Visual Page Builder).

3--НАСТРОЙКА ПЕРЕД ВЕРСТКОЙ
   1. Установить нужные плагины
   1. Отключить Emoji если нет в нем обходимости (см. файл function-Disable-Emoji.php)
   2. Настроить head
   3. Переподключить стили и js файлы в footer и отключить не нужные

3--УСТАНОВКА ПЛАГИНОВ
   1. Определить необходимые плагины
   1. Разместить архивы плагинов в папку TGM/lib/Plugins
   1. Подключить TGM (см. файл function-Load-TGM-plugin.php ).
   2. Подключить плагины в файле theme-plugins.php
   3. Если не используется плагин Yoast SEO, то подключить плагин open graph 

4--НАСТРОЙКА HEAD
   1. В header.php заменить тег с viewport и вставить настройки из файла function-head.php
   2. В function.php добавить настройки из файла function-head.php
   3. Тег <body> добавить сообщение для обновления браузера для старого IE

5--ПОДКЛЮЧЕНИЕ СТИЛЕЙ И JS
   5.1 Добавить функцию в function.php из файла function-inspect_scripts_and_styles.php (Для просмотра подключенных стилей и js)
   5.2 Отключить не нужные и подключенные в head стили и js файлы (см. файл function-scripts-and-styles.php)
   5.3 Подключить в footer стили и js файлы (см. файл function-scripts-and-styles.php)

---ВЕРСТКА
   1. Подключение css и script.js в function.
   2. Подключение атрибутов head и webfontloaded.js в wp_head.
   3. Подключить gulp (yarn/npm init и установить плагины).
   4. Создать главную страницу с ярлыком homepage.
   5. Сделать её статической (Настройки -> чтение).
   6. Создать страницу .php в папку с темой и назвать page-homepage.php (Удалить оттуда функцию комментарии и сайдбар. Подключить контект страницы в get_template_part).
   7. Создать контент страницы в папке template-parts. (content-homepage.php. Удалить оттуда все кроме the_content и entry-content) 
   8. Подключить туда секции
   9. Вставить элементы в header.php

  10. Верстать

--ВЕРСТКА ИЗОБРАЖЕНИЙ
  -  У изображений должны быть дубли меньшего размера для маленьких экранов и большего для мониторов большей плотности
  -  Все изображения должны быть в теге <picture>
  1. С использование плагина LazyLoad
     1.1 К тегу img добавлять класс lazy
     1.2 Вместо src использовать data-lazy-src
     1.3 Вместо srcset использовать data-lazy-srcset
     1.4 Смотреть шаблон в папке plugins/LazyLoad Webp/LazyLoad Webp.jade
  2. Без плагина
     2.1 Делать по шаблону в папке _sections-template/_picture/picture.jade
  3. Верстка через CSS 
     3.1 Делать по шаблону в папке _sections-template/_picture/picture.sass

---ПОДКЛЮЧЕНИЕ WOOCOMERCE
1. Установить плагин WC.
2. Добавить поддержку в function.php
3. Переименовать страницу магазин в каталог
